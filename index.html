02

COATCOAT

Turns a chaotic co-op inbox into a live, self-updating tracker.

COAT authenticates across Gmail and Outlook via OAuth 2.0, then a logistic regression classifier — trained on TF-IDF features from ~1,200 self-labelled emails — sorts every message into offer, interview, or rejection at 94% accuracy and auto-schedules interviews straight onto your calendar. Anything the model isn’t confident about falls back to regex rules instead of failing silently.

  • collaborative filtering surfaces leads from anonymized peer outcomes
  • Gmail + Outlook OAuth 2.0, multi-account on Supabase

03

SignSpeakSignSpeak

Indian Sign Language → speech, in real time.

SignSpeak watches a live camera feed and speaks what it sees. MediaPipe extracts 21 hand landmarks per frame, and I trained a classifier on top of them using a self-curated 26-letter dataset. The hard part isn’t one frame — it’s knowing when a letter ends and a word begins.

  • a 10-frame majority vote kills jitter mid-gesture
  • a background thread treats 1.5s pauses as word breaks, then fires TTS

04

Netflix RatingsNetflix Ratings

IMDb + Rotten Tomatoes, right on the tile.

A Manifest V3 Chrome extension that stamps live IMDb and Rotten Tomatoes scores onto Netflix cards, visible before you even hover. Netflix is an SPA that swaps its DOM constantly, so the trick was a MutationObserver that re-injects on every navigation — no reloads, no flicker.

  • an OMDb background service worker fetches + caches scores
  • MutationObserver survives SPA route changes
